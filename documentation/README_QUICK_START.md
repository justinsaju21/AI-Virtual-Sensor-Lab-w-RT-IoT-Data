## 🚀 One-Click Start

### Windows (Fastest - 30 seconds)
1. Double-click: `start-local.bat`
2. Wait ~60 seconds for both windows to launch
3. Open: `http://localhost:3000`

### macOS/Linux (1 minute)
```bash
cd ~/Desktop/SEM6\ Project/iot-virtual-lab/
chmod +x start-local.sh
./start-local.sh
```

### Manual Setup (All Platforms)
```bash
# Terminal 1 - Backend
cd backend && npm install && npm start

# Terminal 2 - Frontend  
cd frontend && npm install && npm run dev

# Then open: http://localhost:3000
```

## 🐛 Quick Troubleshooting

| Problem | Fix |
|---------|-----|
| Port 5000 in use | `taskkill /IM node.exe /F` (Windows) or kill node process |
| WebSocket won't connect | Check backend running, verify CORS origin |
| Blank dashboard | Open DevTools (F12), check browser console |
| AI not responding | Backend not running or `GEMINI_API_KEY` not set |
| Charts not updating | Refresh browser, check Network tab for WebSocket |

See **LOCAL_HOSTING_SETUP.md** for detailed troubleshooting.
